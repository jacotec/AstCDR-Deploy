# License folder

Drop the **`astcdr.lic`** you received into this folder, then
`docker compose up -d` — this unlocks the **full version**.

Without an `astcdr.lic` here, the **free version** runs
(journal limited to the 100 most recent calls, ingest limited to the last 30 days).

Do **not** commit your `astcdr.lic` (excluded via `.gitignore`).
